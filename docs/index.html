Guzl is an open source viewer for subreddits.

You can view the source code here . I created this for my personal use, but if you'd like to use it, please feel free.

The main concept of Guzl that there is an improved workflow for your sessions. Along with the improved workflow there are no cookies or tracking. Privacy is very important to those of us who love to guzl content.

The core concept of Guzl is the setup. A setup is the order in which you browse your subreddits until you are finished.

Within Guzl you can configure your own setup.

Installation

You can install the dmg here Guzl for Mac.

You can install the exe here Guzl for Windows.

Or you can build from source yourself.

Contribution

You can contribute if you want, or you can just choose to Guzl.

I recommend if you contribute to do it anonymously. Guzl is all about anonymity.
